## Settings :wrench:

Clone the project:
```bash
git clone https://github.com/franciscojsc/jobify.git
```

Enter the folder:
```bash
cd jobify
```

## Deployment :rocket:

Install the dependencies:
```bash
npm install
```

Launch the application:
```bash
npm start
```

Access the application at `http://localhost:3000`.
